# Hospital-Management-System

# Project Overview
A comprehensive relational database system for hospital operations including patient management, doctor schedules, appointments, and medical history tracking.

# Features
- ER Diagram Design
- Database Normalization (1NF, 2NF, 3NF)
- SQL Table Creation & Data Insertion
- Complex SQL Queries (Joins, Subqueries, Views)
